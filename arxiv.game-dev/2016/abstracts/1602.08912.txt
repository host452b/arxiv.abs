Title: Extending Finite Memory Determinacy to Multiplayer Games
Authors: Stéphane Le Roux, Arno Pauly
ID: 1602.08912
URL: https://arxiv.org/abs/1602.08912

Abstract:
We show that under some general conditions the finite memory determinacy of a class of two-player win/lose games played on finite graphs implies the existence of a Nash equilibrium built from finite memory strategies for the corresponding class of multi-player multi-outcome games. This generalizes a previous result by Brihaye, De Pril and Schewe. We provide a number of example that separate the various criteria we explore.
  Our proofs are generally constructive, that is, provide upper bounds for the memory required, as well as algorithms to compute the relevant Nash equilibria.
