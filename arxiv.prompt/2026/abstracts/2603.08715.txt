Title: VeriInteresting: An Empirical Study of Model Prompt Interactions in
Verilog Code Generation
Authors: Luca Collini, Andrew Hennesee, Patrick Yubeaton, Siddharth Garg, Ramesh
Karri
ID: 2603.08715
URL: https://arxiv.org/abs/2603.08715

Abstract:
Rapid advances in language models (LMs) have created new opportunities for
automated code generation while complicating trade-offs between model
characteristics and prompt design choices. In this work, we provide an empirical
map of recent trends in LMs for Verilog code generation, focusing on
interactions among model reasoning, specialization, and prompt engineering
strategies. We evaluate a diverse set of small and large LMs, including general-
purpose, reasoning, and domain-specific variants. Our experiments use a
controlled factorial design spanning benchmark prompts, structured outputs,
prompt rewriting, chain-of-thought reasoning, in-context learning, and
evolutionary prompt optimization via Genetic-Pareto. Across two Verilog
benchmarks, we identify patterns in how model classes respond to structured
prompts and optimization, and we document which trends generalize across LMs and
benchmarks versus those that are specific to particular model-prompt
combinations.
