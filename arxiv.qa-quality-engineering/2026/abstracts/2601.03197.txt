Title: Software-Defined Agentic Serving
Authors: Saurabh Agarwal, Marco Laju, Jayanth Srinivasa, Myungjin Lee, Aditya Akella
ID: 2601.03197
URL: https://arxiv.org/abs/2601.03197

Abstract:
As multi-agent LLM pipelines grow in complexity, existing serving paradigms fail
to adapt to the dynamic serving conditions. We argue that agentic serving
systems should be programmable and system-aware, unlike existing serving which
statically encode the parameters. In this work, we propose a new SDN-inspired
agentic serving framework that helps control the key attributes of communication
based on runtime state. This architecture enables serving-efficient, responsive
agent systems and paves the way for high-level intent-driven agentic serving.
