Title: The test set for the TransCoder system
Authors: Ernest Davis
ID: 2008.00293
URL: https://arxiv.org/abs/2008.00293

Abstract:
The TransCoder system translates source code between Java, C++, and Python 3.
The test set that was used to evaluate its quality is missing important features
of Java, including the ability to define and use classes and the ability to call
user-defined functions other than recursively. Therefore, the accuracy of
TransCoder over programs with those features remains unknown.
