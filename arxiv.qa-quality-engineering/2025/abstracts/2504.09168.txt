Title: SBFT Tool Competition 2025 -- Java Test Case Generation Track
Authors: Fitsum Kifetew, Lin Yun, Davide Prandi
ID: 2504.09168
URL: https://arxiv.org/abs/2504.09168

Abstract:
This short report presents the 2025 edition of the Java Unit Testing Competition
in which four test generation tools (EVOFUZZ, EVOSUITE, BBC, and RANDOOP) were
benchmarked on a freshly selected set of 55 Java classes from six different open
source projects. The benchmarking was based on structural metrics, such as code
and mutation coverage of the classes under test, as well as on the readability
of the generated test cases.
