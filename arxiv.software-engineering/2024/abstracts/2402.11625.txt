Title: SpeCrawler: Generating OpenAPI Specifications from API Documentation Using Large Language Models
Authors: Koren Lazar, Matan Vetzler, Guy Uziel, David Boaz, Esther Goldbraich et al.
ID: 2402.11625
URL: https://arxiv.org/abs/2402.11625

Abstract:
In the digital era, the widespread use of APIs is evident. However, scalable
utilization of APIs poses a challenge due to structure divergence observed in
online API documentation. This underscores the need for automatic tools to
facilitate API consumption. A viable approach involves the conversion of
documentation into an API Specification format. While previous attempts have
been made using rule-based methods, these approaches encountered difficulties in
generalizing across diverse documentation. In this paper we introduce
SpeCrawler, a comprehensive system that utilizes large language models (LLMs) to
generate OpenAPI Specifications from diverse API documentation through a
carefully crafted pipeline. By creating a standardized format for numerous APIs,
SpeCrawler aids in streamlining integration processes within API orchestrating
systems and facilitating the incorporation of tools into LLMs. The paper
explores SpeCrawler's methodology, supported by empirical evidence and case
studies, demonstrating its efficacy through LLM capabilities.
