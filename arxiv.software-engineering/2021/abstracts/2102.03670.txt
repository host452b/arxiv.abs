Title: Recommending More Efficient Workflows to Software Developers
Authors: Dylan Bates
ID: 2102.03670
URL: https://arxiv.org/abs/2102.03670

Abstract:
Existing recommendation systems can help developers improve their software
development abilities by recommending new programming tools, such as a
refactoring tool or a program navigation tool. However, simply recommending
tools in isolation may not, in and of itself, allow developers to successfully
complete their tasks. In this paper, I introduce a new recommendation system
that recommends workflows, or sequences of tools, to developers. By learning
more efficient workflows, the system could make software developers more
efficient.
