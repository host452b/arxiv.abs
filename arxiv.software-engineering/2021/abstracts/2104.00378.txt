Title: The Comprehensive Blub Archive Network: Towards Design Principals for Open Source Programming Language Repositories
Authors: Seamus Brady
ID: 2104.00378
URL: https://arxiv.org/abs/2104.00378

Abstract:
Many popular open source programming languages (Perl, Ruby or Python for
example) have systems for distributing packaged source code that software
developers can use when working in that particular programming language. This
paper will consider the design principals that should be followed if designing
such an open source code repository.
