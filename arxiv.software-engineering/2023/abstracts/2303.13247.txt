Title: Optimizing Duplicate Size Thresholds in IDEs
Authors: Konstantin Grotov, Sergey Titov, Alexandr Suhinin, Yaroslav Golubev, Timofey Bryksin
ID: 2303.13247
URL: https://arxiv.org/abs/2303.13247

Abstract:
In this paper, we present an approach for transferring an optimal lower size
threshold for clone detection from one language to another by analyzing their
clone distributions. We showcase this method by transferring the threshold from
regular Python scripts to Jupyter notebooks for using in two JetBrains IDEs,
Datalore and DataSpell.
