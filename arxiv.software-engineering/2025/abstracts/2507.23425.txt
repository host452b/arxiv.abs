Title: Dynamic and Static Analysis of Python Software with Kieker Including Reconstructed Architectures
Authors: Daphné Larrivain, Shinhyung Yang, Wilhelm Hasselbring
ID: 2507.23425
URL: https://arxiv.org/abs/2507.23425

Abstract:
The Kieker observability framework is a tool that provides users with the means
to design a custom observability pipeline for their application. Originally
tailored for Java, supporting Python with Kieker is worthwhile. Python's
popularity has exploded over the years, thus making structural insights of
Python applications highly valuable. Our Python analysis pipeline combines
static and dynamic analysis in order to build a complete picture of a given
system.
