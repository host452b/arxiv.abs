Title: A Purpose-oriented Study on Open-source Software Commits and Their Impacts on Software Quality
Authors: Jincheng He, Zhongheng He
ID: 2503.02232
URL: https://arxiv.org/abs/2503.02232

Abstract:
Developing software with the source code open to the public is prevalent;
however, similar to its closed counter part, open-source has quality problems,
which cause functional failures, such as program breakdowns, and non-functional,
such as long response times. Previous researchers have revealed when, where, how
and what developers contribute to projects and how these aspects impact software
quality. However, there has been little work on how different categories of
commits impact software quality. To improve open-source software, we conducted
this preliminary study to categorize commits, train prediction models to
automate the classification, and investigate how commit quality is impacted by
commits of different purposes. By identifying these impacts, we will establish a
new set of guidelines for committing changes that will improve the quality.
