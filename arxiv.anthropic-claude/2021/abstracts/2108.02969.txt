Title: How the Analyzer can Help the User Help the Analyzer
Authors: Yannick Moy
ID: 2108.02969
URL: https://arxiv.org/abs/2108.02969

Abstract:
The automation offered by modern program proof tools goes hand in hand with the
capability to interact with the tool when the verification fails. The SPARK
proof tool tries to help the user by providing the right information, so that
the user can help the tool complete the proof. In this article, we present these
mechanisms and how they work concretely on a simple running example.
