Title: Microstructure Effects on Daily Return Volatility in Financial Markets
Authors: Andreas Krause
ID: cond-mat/0011295
URL: https://arxiv.org/abs/cond-mat/0011295

We simulate a series of daily returns from intraday price movements initiated by
microstructure elements. Significant evidence is found that daily returns and
daily return volatility exhibit first order autocorrelation, but trading volume
and daily return volatility are not correlated, while intraday volatility is. We
also consider GARCH effects in daily return series and show that estimates using
daily returns are biased from the influence of the level of prices. Using daily
price changes instead, we find evidence of a significant GARCH component. These
results suggest that microstructure elements have a considerable influence on
the return generating process.
