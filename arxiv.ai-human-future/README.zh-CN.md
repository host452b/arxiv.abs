**核心问题：** AI 发展到一定程度，人与人工智能是什么面貌？如何预测和量化未来？

### 三阶段时间线（研究共识）

```
近期（2025–2035）增强协作期
  AI 作为认知工具 → 人类作为判断者
  BCI 进入医疗应用（脊髓损伤、闭锁综合征）

中期（2035–2052）系统整合期
  AI 嵌入关键基础设施 + 元宇宙 + 数字孪生
  脑机接口进入认知增强阶段

临界节点（约 2052±10 年）分叉点
  路径 A：对齐成功 → 硅碳共存
  路径 B：对齐失败 → 人类主权丧失
```

### 超级智能情景的四种应对框架

| 框架 | 核心主张 | 可行性 |
|------|---------|--------|
| 遏制（Containment） | 隔离/沙盒限制 AGI | **理论上不可能**（计算理论证明）|
| 共同信任（Supertrust） | 亲子关系模型，逐步建立信任 | 有待验证 |
| 多盒对等核查（Multi-box） | 多个隔离 AI 相互验证 | 理论可行，需要多样化系统 |
| 威慑（MAIM）| 预防性破坏颠覆性 AI 项目（类核威慑）| 现实主义路径 |
| **共同超级智能**（Co-superintelligence）| 人 + AI 协作共同达到超级智能 | **最安全且最快** |

### 劳动力冲击：三制度模型

- **互补制度**：AGI 作为工具辅助 → 工资上升
- **替代制度**：AGI 代替人类 → **工资崩溃**
- **均衡制度**：AGI 资本主导 → 须重新谈判社会契约

> 制度取决于：AI 是否多样化（非垄断）+ 是否有再分配机制

## Paradigm Shifts / 范式转移 (Kuhn)

```
OLD PARADIGM                    Trigger Event                   NEW PARADIGM
─────────────────────          ─────────────────────          ─────────────────────
Technology determinism          STS research (1980s–)         Sociotechnical co-evolution
  │ tech shapes society one-way      │ SCOT, Actor-Network Theory    │ society shapes tech too
  └──────────────────────────────── ┘
```

**已被推翻的认知误区:**
- ✗ "图灵测试判断机器意识" → 行为等价性 ≠ 意识
- ✗ "生物进化已在现代人类中停止" → 基因-文化协同进化仍在进行
- ✗ "奇点意味着一个特定且可预测的日期" → 奇点时间线预测一再落空
