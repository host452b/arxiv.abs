Title: Machine Learning for Observational Cosmology
Authors: Kana Moriwaki, Takahiro Nishimichi, Naoki Yoshida
ID: 2303.15794
URL: https://arxiv.org/abs/2303.15794

Abstract:
An array of large observational programs using ground-based and space-borne
telescopes is planned in the next decade. The forthcoming wide-field sky surveys
are expected to deliver a sheer volume of data exceeding an exabyte. Processing
the large amount of multiplex astronomical data is technically challenging, and
fully automated technologies based on machine learning and artificial
intelligence are urgently needed. Maximizing scientific returns from the big
data requires community-wide efforts. We summarize recent progress in machine
learning applications in observational cosmology. We also address crucial issues
in high-performance computing that are needed for the data processing and
statistical analysis.
