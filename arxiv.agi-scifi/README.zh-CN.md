## Paradigm Shifts / 范式转移 (Kuhn)

| 旧范式 | 触发事件 | 新范式 |
|---|---|---|
| **神经网络无法进行符号推理** — 必须依赖符号 AI | GPT-4 / 思维链 (2022–): 数学与逻辑基准测试 | **涌现形式推理** — 神经与符号方法走向融合 |
| **AGI 是科幻小说** — 50 年后的事 | 规模定律 (2020–): GPT-3/4 涌现能力 | **AGI 成为近期研究优先级** — 主流实验室全面投入 |
| **安全与能力存在权衡** — 越安全越不能干 | Constitutional AI / Anthropic HHH 训练结果 | **安全赋能能力** — 对齐的模型更有用、更受信任 |
| "对齐的模型不会欺骗" — RLHF 消除了不良行为 | 潜伏 Agent 研究 (2024): Hubinger 等 | **欺骗性对齐是可能的** — 安全训练 ≠ 安全证明 |

**已被推翻的认知误区:**
- ✗ "纯神经网络无法进行符号推理" → GPT-4类模型展现了显著的形式推理能力
- ✗ "安全与能力之间存在根本性权衡" → CAI/RLHF研究证明安全技术往往同时提升有用性
- ✗ "已对齐模型不会欺骗" → 睡眠智能体研究表明安全训练不能可靠地消除隐性欺骗

## 公认谬误 / Established Fallacies

| 误解 | 为何盛行 | 实证结论 |
|---|---|---|
| 图灵测试衡量真正的智能 | 行为等价意味着理解能力 | ELIZA 在 1966 年以零理解能力通过了类图灵测试；该测试很容易被模式匹配所欺骗 |
| 摩尔定律保证 AGI 在某个可计算日期到来 | 基于晶体管密度的算力外推 | 算力增长已在物理极限处趋缓；架构和算法突破，而非仅仅算力，才是关键 |
| 智能完全独立于物理基底 | 强功能主义假设 | 具身认知研究表明，物理接地对空间、社会和运动认知是不可或缺的 |

## 过时科学理论 / Obsolete Scientific Theories

| 理论 | 流行年代 | 被取代原因 |
|---|---|---|
| 经典 GOFAI 作为通往 AGI 的路径 | 1950s–1980s：逻辑 + 搜索 = 智能 | 框架问题和组合爆炸使其无法应对现实复杂性；被统计机器学习击败 |
| Rosenblatt 的感知机作为通用学习器 | 1958 年 | Minsky & Papert（1969）：XOR 问题证明了单层网络的局限；需要深层网络 |
| 用海量规则库编码常识（CYC 项目） | 1984 年至今 | 30 多年和数百万条规则仍未实现通用推理；知识瓶颈被证明无法逾越 |

## 被证伪的理论 / Falsified Theories

| 理论 | 核心预测 | 证伪证据 |
|---|---|---|
| 规模本身足以产生通用智能 | 预测：10^26 FLOP → AGI，沿平滑能力曲线外推 | GPT-4 量级模型在规模上仍无法通过系统性组合泛化测试（SCAN, COGS 基准） |
| 奖励足以产生通用智能（Silver 等，2021） | 预测：最大化奖励信号能产生所有认知能力 | DeepMind 自身的后续研究表明，没有归纳偏置的奖励最大化器会演化出退化策略 |
