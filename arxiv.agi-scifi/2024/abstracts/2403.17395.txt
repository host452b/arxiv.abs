Title: An Open-source End-to-End Logic Optimization Framework for Large-scale
Boolean Network with Reinforcement Learning
Authors: Zhen Li, Kaixiang Zhu, Xuegong Zhou, Lingli Wang
ID: 2403.17395
URL: https://arxiv.org/abs/2403.17395

Abstract:
We propose an open-source end-to-end logic optimization framework for large-
scale boolean network with reinforcement learning.
