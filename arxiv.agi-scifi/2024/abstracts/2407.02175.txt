Title: Interaction spaces: towards a universal mathematical theory of complex
systems
Authors: Paolo Giordano
ID: 2407.02175
URL: https://arxiv.org/abs/2407.02175

Abstract:
We present the first steps of interaction spaces theory, a universal
mathematical theory of complex systems which is able to embed cellular automata,
agent based models, master equation based models, stochastic or deterministic,
continuous or discrete dynamical systems, networked dynamical models, artificial
neural networks and genetic algorithms in a single notion. Therefore,
interaction spaces represent a common mathematical language that can be used to
describe several complex systems modeling frameworks. This is the first step to
start a mathematical theory of complex systems. Every notion is introduced both
using an intuitive description by listing lots of examples, and using a modern
mathematical language.
