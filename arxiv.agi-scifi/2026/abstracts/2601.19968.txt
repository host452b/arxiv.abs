Title: What is the AGI in Offensive Security ?
Authors: Youngwoong Cho
ID: 2601.19968
URL: https://arxiv.org/abs/2601.19968

Abstract:
What is the AGI in Offensive Security? One can break it down into two questions
: (1) any offensive security tasks could be reduced into symbolic language
manipulation (language representation + reasoning), (2) powerful language model
(LLM) are enough to "deal with" any symbolic language manipulation. This paper
can formally model a target system as a state machine and a hacker as an
interactive symbolic agent. And it shows that every interaction in an offensive
engagement can be encoded as a finite string. This paper provides definitions,
short lemmas, and open discussion.
