Title: Trust, But Verify: An Empirical Evaluation of AI-Generated Code for SDN
Controllers
Authors: Felipe Avencourt Soares, Muriel F. Franco, Eder J. Scheid, Lisandro Z.
Granville
ID: 2510.20703
URL: https://arxiv.org/abs/2510.20703

Abstract:
Generative Artificial Intelligence (AI) tools have been used to generate human-
like content across multiple domains (e.g., sound, image, text, and
programming). However, their reliability in terms of correctness and
functionality in novel contexts such as programmable networks remains unclear.
Hence, this paper presents an empirical evaluation of the source code of a POX
controller generated by different AI tools, namely ChatGPT, Copilot, DeepSeek,
and BlackBox.ai. To evaluate such a code, three networking tasks of increasing
complexity were defined and for each task, zero-shot and few-shot prompting
techniques were input to the tools. Next, the output code was tested in emulated
network topologies with Mininet and analyzed according to functionality,
correctness, and the need for manual fixes. Results show that all evaluated
models can produce functional controllers. However, ChatGPT and DeepSeek
exhibited higher consistency and code quality, while Copilot and BlackBox.ai
required more adjustments.
