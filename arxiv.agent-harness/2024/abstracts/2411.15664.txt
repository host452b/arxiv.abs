Title: Enabling Efficient Serverless Inference Serving for LLM (Large Language Model) in the Cloud
Authors: Himel Ghosh
ID: 2411.15664
URL: https://arxiv.org/abs/2411.15664

Abstract:
This review report discusses the cold start latency in serverless inference and
existing solutions. It particularly reviews the ServerlessLLM method, a system
designed to address the cold start problem in serverless inference for large
language models. Traditional serverless approaches struggle with high latency
due to the size of LLM checkpoints and the overhead of initializing GPU
resources. ServerlessLLM introduces a multitier checkpoint loading system,
leveraging underutilized GPU memory and storage to reduce startup times by 6--8x
compared to existing methods. It also proposes live inference migration and a
startup-time-optimized model scheduler, ensuring efficient resource allocation
and minimizing delays. This system significantly improves performance and
scalability in serverless environments for LLM workloads. Besides ServerlessLLM,
several other methods from recent research literature, including Rainbowcake,
are reviewed in this paper. Further discussions explore how FaaS providers
tackle cold starts and the possible future scopes.
