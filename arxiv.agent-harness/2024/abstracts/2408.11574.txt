Title: Drama Engine: A Framework for Narrative Agents
Authors: Martin Pichlmair, Riddhi Raj, Charlene Putney
ID: 2408.11574
URL: https://arxiv.org/abs/2408.11574

Abstract:
This technical report presents the Drama Engine, a novel framework for agentic
interaction with large language models designed for narrative purposes. The
framework adapts multi-agent system principles to create dynamic, context-aware
companions that can develop over time and interact with users and each other.
Key features include multi-agent workflows with delegation, dynamic prompt
assembly, and model-agnostic design. The Drama Engine introduces unique elements
such as companion development, mood systems, and automatic context summarising.
It is implemented in TypeScript. The framework's applications include
multi-agent chats and virtual co-workers for creative writing. The paper
discusses the system's architecture, prompt assembly process, delegation
mechanisms, and moderation techniques, as well as potential ethical
considerations and future extensions.
